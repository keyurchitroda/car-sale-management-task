# Car Sales & Commission Management System

## Prerequisites

- Node.js
- PostgreSQL
- npm

## Environment Setup

Create a `.env` file in the backend root directory:

PORT=3000  
DB_HOST=localhost  
DB_PORT=5432  
DB_USER=postgres  
DB_PASSWORD=your_db_password  
DB_NAME=car_management

## Install Dependencies

npm install

## Run Migration Files

npm run migrate:latest

## Run Seeder (Salesman)

npm run seed:latest

## Run the Application

npm run start

## Application URL

http://localhost:3000
